# 스피드체크 — 마케팅 2차 플랜 (Phase M1~M4)

기준: PRD v1.0 §2(KPI)·§8(릴리즈), 배포 완료 상태(speed-value.com). 목표는 Phase 1 비즈니스 목표인 **3개월 내 MAU 3만 + 첫 제휴 개통**.

한국 시장 전제: 검색 유입의 절반 이상이 네이버 → **구글 SEO와 네이버를 별도 트랙**으로 운영한다.

## M2. 콘텐츠 SEO (주 2~8) — 검색 수요 선점

PRD의 "SEO 기초 콘텐츠 20편"을 `/guide/*` 라우트로 본체에 직접 게시(서버 렌더링). 키워드 클러스터별 배분:

| 클러스터    | 예시 키워드                                                         | 편수 | 페르소나                      |
| ----------- | ------------------------------------------------------------------- | ---- | ----------------------------- |
| 측정 하우투 | 인터넷 속도 측정 방법, 와이파이 속도 측정, 노트북/폰 속도 측정      | 4    | 전체                          |
| 문제 해결   | 인터넷 갑자기 느려짐, 와이파이 느릴 때, 유튜브 버퍼링, 핑 낮추는 법 | 6    | ① 불만 사용자, ③ 게이머       |
| 통신사 비교 | KT vs SK vs LG 인터넷, 통신사별 속도, 500메가 vs 1기가 체감         | 4    | ② 이사·개통                   |
| 이사·개통   | 이사 인터넷 이전 vs 신규, 인터넷 현금 사은품 구조, 약정 위약금      | 3    | ② 이사·개통 (리드젠 직결)     |
| 장비        | 공유기 위치·교체 시기, 메시 와이파이, 5GHz vs 2.4GHz                | 3    | ④ 와이파이 문제 (커머스 직결) |

운영 원칙:

- 모든 글 하단에 측정 위젯/CTA 임베드 → 글 유입을 측정으로 전환
- FAQ·HowTo 구조화 데이터 적용, 글 간 내부링크
- 주 2~3편 페이스. 실측 데이터 기반 주장만 (측정 정확도 시비 리스크 §10 대응)

**네이버 트랙 (병행)**: 네이버 웹문서만으론 노출 한계 → 네이버 블로그 개설, 가이드 콘텐츠 요약본 게시 + 본체 링크. 지식iN "인터넷 느려요" 질문에 성실 답변 + 링크(과도한 홍보 금지, 계정 신뢰도 우선).

## M3. 배포·홍보 채널 (주 3~12) — 초기 트래픽 점화

SEO는 3~6개월 걸린다. 그동안 직접 유입을 만든다.

**커뮤니티 (순서대로, 스팸으로 안 보이게)**

1. 뽐뿌 인터넷/통신 게시판, 클리앙 — "무료·무가입 속도측정 사이트 만들었습니다" 제작기 형식이 광고보다 반응 좋음
2. 디시 통신/컴퓨터 갤러리, 에펨코리아, 루리웹(게이머 — 핑·지터 소구)
3. GeekNews·디스콰이엇 — 제작기/기술 스택 공유, 개발자 백링크
4. 반응 좋은 글의 피드백 → 제품 개선 루프

**바이럴 루프 (FR-8 고도화)**

- 결과 이미지 카드 생성(캔버스) + 카카오톡 SDK 공유 구현 — 공유 1회가 광고비 0원의 유입
- 공유 URL에 `?ref=share` UTM → `result_shared` → 재유입 전환 측정

**위젯 (FR-14, 백링크 엔진)**

- iframe 임베드 위젯 + "스피드체크 제공" 백링크
- 타깃: 부동산/자취 카페, 인테리어 블로그, IT 블로거 — 도메인 권위 상승 + 상시 유입

**시즌 이벤트 대응**

- 통신사 장애 발생 시 = 트래픽 골든타임. 장애 현황판(FR-15) 전까지는 수동으로라도 "지금 KT 느린가요?" 콘텐츠를 장애 당일 게시 → 실시간 검색 수요 흡수

## KPI 대시보드 (주간 체크)

- 유입: 오가닉 세션(구글/네이버 분리), 커뮤니티 유입, 공유 재유입
- 품질: 측정 완료율 ≥ 90%, LCP ≤ 1.5초
- 전환: `verdict_shown` → `compare_cta_click` CTR, 리드 수
- 콘텐츠: 색인 페이지 수, 키워드 순위(인터넷 속도 측정 top10 목표)
